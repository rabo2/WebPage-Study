new Object 와 {} 생성

    let obj1 = new Object();  //new 연산자를 이용해 객체생성
    let obj2 = {};           //객체 리터럴로 객체 생성(내부적으로 new Object호출)

    리터럴 객체 생성을 권창하는 이유
    1. 코딩 시 가독성이 좋음
    2. 속도가 빠름
    3. JavaScript의 오버라이딩을 방지(new Object()는 오버라이딩이 가능하므로 위험)

    
new Array와 []생성


    let arr = new Array();
    let arr = [];

    직관적인 리터럴 객체생성을 더 선호 한다.

    
배열과 객체의 차이


    배열 : []에 직접 선언할 수 있따.
    const info = ['a', 'b' ,'c','d'];

    객체 : {}안에 선언할 수 없고 key:value값으로 이루어져 있다.
    const info = {

    }
