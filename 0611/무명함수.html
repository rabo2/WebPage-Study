무명함수 = 익명함수 : 함수 명이 없는 함수
    단 한번 사용되는 함수에 주로 사용된다.
    코드의 간결함과 중복될 수 있는 함수명을 피할 수 있다.

    ===================================================

    특정 변수에 대입하는 무명(익명)함수 = 함수표현식
    : 생성된 함수를 변수에 할당하여 사용한다.
    ex)
    var sum = function(a,b){
      var m = a+b;
      return m;
    }
    console. log(sum(1,2)); //결과 3
