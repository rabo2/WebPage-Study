선언식 함수(=기명함수, Named Function) : 브라우저 런타임 이전에 선언되는 함수
    표현식 함수(=dlraudgkatn, Annoymouse Function) : 브라우저 런타임에 동적으로 선언되는 함수
    람다 함수(ES6, java8버전에서 나온 함소, 화살표 함수)
    : var 함수명 = (매개변수)=>{실행문}
    매개변수가 하나만 있으면 괄호 생략 가능
    매개변수가 없거나 여러개일 경우 괄화 안에 지정한다.
    람다함수 내에서 this(현재 수행중인 객체)를 사용할 수 없다. - this는 람다 자체가 된다.
