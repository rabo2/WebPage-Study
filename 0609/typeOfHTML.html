typeof : 변수의 데이터 타입을 반환하는 연산자

      원시타입(prmitive type)
      - 숫자(Number) : 데이터 타입이 수일 때
      - 문자열(String) : 데이터 타입이 문자열일 때
      - 논리형(Boolean) : 데이터 타입이 boolean일 때
      - Null
      - Undefined
      - symbol : 데이터 타입이 심볼일 때

      참조타입(Reference type) - 원본이 바뀌면 복사본도 똑같이 바뀐다.
      -  객체, 배열, 함수(Object) : 데이터 타입이 함수, 배열 등 객체일 때

      ==============================================
      es6에서 새롭게 추가된 7번째 원시타입 symbol
      변경 불가능한 원시 타입의 값
      다른 값에 중복되지 않는 유일무이한 값
      이름의 충돌 위험이 있는 객체의 프로퍼티키를 만들기 위해 사용
